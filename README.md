### **Traditional Development Setup**
```bash
git clone https://github.com/antoniogomezgallardo/devToolsKit.git
cd devToolsKit
npm install && npm run playwright:install
npm run dev  # http://localhost:1234
```

### **🚀 Power User Setup (Recommended)**
